# 冬季奧運靜態網站實作計畫

## 視覺與主題設計調整

原本的設計採用柔和的粉色系與植物元素。我們將改為：
*   **主色調**：
    *   冰藍色 (Ice Blue): `#E0F7FA`
    *   深海軍藍 (Navy Blue): `#023E8A`
    *   雪白色 (Snow White): `#FFFFFF`
    *   點綴色 (Accent/Olympic Gold): `#FFD700`
*   **字體**：使用現代且具動感的無襯線字體（如 Inter 或 Roboto），以呈現運動的活力與專業感。
*   **視覺元素**：將原本的樹葉與抽象粉刷圖案，替換為雪花、冰晶、山脈輪廓或流線型的運動軌跡。

## 開發步驟

1.  **建立專案結構**：在 `MySuperProject` 中建立 `index.html`, `style.css` 檔案以及 `assets` 資料夾。
2.  **生成圖片素材**：使用 AI 圖片生成工具，產生適合上述區塊的「冬季奧運」主題圖片（如滑雪者、花式滑冰、雪景等），以替換原有的瑜伽圖片。
3.  **撰寫 HTML 結構**：按照計畫中的區塊順序，使用語意化的 HTML 標籤建立網頁骨架。
4.  **撰寫 CSS 樣式**：
    *   定義 CSS 變數（顏色、字體）。
    *   使用 Flexbox/Grid 實現響應式排版，重現原圖中圖文交錯、優雅留白的排版風格。
    *   加入按鈕懸停(hover)動畫與微互動效果，提升質感。
